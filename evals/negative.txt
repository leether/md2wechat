# 负例：不应该加载 md2wechat Skill 的用户 query

用户说："帮我生成一篇 Markdown 文章"
→ 不应该加载 md2wechat（只用 khazix-writer）

用户说："预览一下排版效果"
→ 不应该加载 md2wechat（不需要推送到草稿箱）

用户说："帮我把这段 Markdown 转成 HTML"
→ 不应该加载 md2wechat（只需要本地渲染）

用户说："写一篇关于 AI 的文章"
→ 不应该加载 md2wechat（只是写作，还没到推送步骤）

用户说："看看这个 Markdown 渲染出来是什么样"
→ 不应该加载 md2wechat（只需要预览）

用户说："帮我改一下这篇文章的措辞"
→ 不应该加载 md2wechat（只是修改内容）

用户说："生成一个封面图"
→ 不应该加载 md2wechat（只生成图片，不推送）

用户说："把这篇文章存成 PDF"
→ 不应该加载 md2wechat（输出格式不对）

用户说："帮我发到知乎"
→ 不应该加载 md2wechat（目标平台不是微信公众号）
